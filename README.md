# Task 1: Caesar Cipher

## Description
This is a Python program that encrypts and decrypts messages using the Caesar Cipher.  
- You can choose to encrypt or decrypt a message.  
- You can enter any text message.  
- You can specify a shift number for the cipher.  
- After running, it shows the result along with a confirmation message.

## How to Run
1. Make sure you have **Python 3** installed.  
2. Open the terminal and go to the folder containing `caesar_cipher.py`.  
3. Run the program:
   ```bash
   python3 caesar_cipher.py

Follow the instructions on the screen:

Enter 1 to encrypt or 2 to decrypt.

Type your message.

Enter the shift number.

The program will show the encrypted or decrypted message.

Enter choice (1 for Encrypt / 2 for Decrypt): 1
What is your secret message? THANKYOU
Enter your shift number (e.g., 3): 3
Encrypted message: WKDQNBRX
